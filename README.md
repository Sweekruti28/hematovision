### Advanced Blood Cell Classification Using Transfer Learning

HematoVision is a deep learning-based web application designed to automatically classify blood cell images using **Transfer Learning**. The system leverages a pre-trained **MobileNetV2 Convolutional Neural Network (CNN)** to accurately identify different types of blood cells.

This project demonstrates how modern computer vision techniques can assist healthcare professionals by improving diagnostic efficiency and reducing manual workload.

Blood cell classification plays a critical role in medical diagnostics. Manual identification is time-consuming and requires expertise. HematoVision automates this process by utilizing deep learning and transfer learning techniques.

- **Eosinophil**
- **Lymphocyte**
- **Monocyte**
